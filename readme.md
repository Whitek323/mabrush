#### Pattern collect data of filename

{ข้อมูลชุดที่}.{ซ้าย,กลาง,ขวา}-{ฟันบน,ฟันล่าง}-{นอก,ผิวฟัน,ใน} 

{i++}.{1,2,3}-{1,2}-{1,2,3}

ex. 16 pattern

0.1-1-1 = ซ้าย บน ด้านนอก

0.1-1-2 = ซ้าย บน ผิวฟัน

0.1-1-3 = ซ้าย บน ด้านใน

0.1-2-1 = ซ้าย ล่าง ด้านนอก

0.1-2-2 = ซ้าย ล่าง ผิวฟัน

0.1-2-3 = ซ้าย ล่าง ด้านใน

0.2-1-1 = กลาง บน ด้านนอก 

0.2-1-3 = กลาง บน ด้านใน

0.2-2-1 = กลาง ล่าง ด้านนอก 

0.2-2-3 = กลาง ล่าง ด้านใน

0.3-1-1 = ขวา บน ด้านนอก

0.3-1-2 = ขวา บน ผิวฟัน

0.3-1-3 = ขวา บน ด้านใน

0.3-2-1 = ขวา ล่าง ด้านนอก

0.3-2-2 = ขวา ล่าง ผิวฟัน

0.3-2-3 = ขวา ล่าง ด้านใน

> ตั้งชื่อ Folder ที่เก็บไฟล์นี้ว่าใช้มือซ้ายหรือมือขวา
